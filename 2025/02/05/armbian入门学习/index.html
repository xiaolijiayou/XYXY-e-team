armbian常用命令

命令 功能 说明
armbian-update 更新系统内核软件包
sudo armbian-software 安装常用软件
armbian-config 修改系统配置 GUI界面
nmtui 网络管理TUI 使用链接
rmbian-swap 创建swap分区
rmbian-tf 激活tf,sd,u盘的剩余空间
rmbian-sync 更新本地计算机所有服务脚本
udo nano /etc/docker/daemon.json 以管理员身份编辑deamon.json文件
rmbian-install 安装 Armbian 到 EMMC
rmbian-led 控制 LED 显示
rmbian-ddbr 备份/还原 EMMC 原系统
ocker --version 查询docker版本

附录B：Armbian系统网络配置示例

在/etc/network/interfaces​文件中添加以下内容，可以配置一个静态IP地址：
auto eth0
iface eth0 inet static
address 192.168.1.100
netmask 255.255.255.0
gateway 192.168.1.1

保存文件后，使用sudo /etc/init.d/networking restart​命令重启网络服务，使配置生效。

问题集合

Q1:安装软件出现docker无法联网报错

​​

解决：原因是docker从国外源站拉取，你的设备不能正常访问外网；2024年9月后国内的大学docker镜像仓库大都停止服务了，因此更换国内源也不行。

sudo nano /etc/docker/daemon.json
修改配置文件 /etc/docker/daemon.json，修改后内容如下：
{
“runtimes”: {
“nvidia”: {
“args”: [],
“path”: “nvidia-container-runtime”
}
},
“registry-mirrors”: [“https://docker.registry.cyou“,
“https://docker-cf.registry.cyou“,
“https://dockercf.jsdelivr.fyi“,
“https://docker.jsdelivr.fyi“,
“https://dockertest.jsdelivr.fyi“,
“https://mirror.aliyuncs.com“,
“https://dockerproxy.com“,
“https://mirror.baidubce.com“,
“https://docker.m.daocloud.io“,
“https://docker.nju.edu.cn“,
“https://docker.mirrors.sjtug.sjtu.edu.cn“,
“https://docker.mirrors.ustc.edu.cn“,
“https://mirror.iscas.ac.cn“,
“https://docker.rainbond.cc"]
}
